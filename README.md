# Timeout Test

A platformer game created with MiniDev ONE Template.

## Quick Start

```bash
npm install
npm run dev
```

## ONE Template

This project uses the **MiniDev ONE Template** - one template, infinite possibilities.

All configuration in `src/lib/config.ts`.

### Configuration
- **Type**: game
- **Category**: platformer
- **Difficulty**: medium
- **Players**: solo
